Setup
=====

Install skydive in the path. 
Prebuilt binaires are available at https://github.com/skydive-project/skydive-binaries.

Install terraform 0.8.

Install jq >= 0.3.

Source openstack credentials for lab2.

Usage
=====

Run all tests:

    ./run.sh

Run a specific test:

    ./run.sh sg

or

    cd tests/sg/
    ./test.sh
